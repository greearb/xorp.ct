What is Xorp?

Xorp (pronounced z-o-r-p)is an open router software platform being developed primarily at the International Computer Science Institute in Berkeley. The goal is to produce a complete set of router software that is stable enough and fully featured enough for production use, initially on edge routers, but also due to our focus on extensibility, to be an excellent platform for Internet research.

Getting Xorp

Xorp is free software. As of June 2003, Xorp is alpha release research software. This means it's not ready for use as a router, but has reached the stage where the architecture is basically complete and it might be used as the basis for research. See the current status of the different parts of Xorp, or the release notes for information about changes between releases.

  • Release 0.3, an alpha version of Xorp, is available for download.
  • The Xorp source code is accessible through our anonymous CVS server.
  • Individual files and changes to files can be viewed through a CVSweb interface.

Please note that to build all the Xorp modules and test programs with debugging enabled, you will need nearly 1GB of free disk space. Xorp is written in C++, and has been developed on FreeBSD 4.x and Redhat Linux 7.3. The current release 0.3 is known to compile with gcc 2.95.x, 2.96, and 3.2.1.
