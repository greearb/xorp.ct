What is XORP?

XORP is the eXtensible Open Router Platform. The XORP project is developing an open source software router. The software is intended to be stable and fully featured enough for production use, and flexible and extensible enough for research use.

XORP runs natively on FreeBSD and Linux and is largely platform independent. We hope to be able to support additional platforms and the Click Modular Router as a forwarding path in future.

The XORP project is funded by Intel and the NSF and employs a small team of developers at the International Computer Science Institute in Berkeley.

Getting XORP

XORP is free software. As of August 2003, XORP is alpha release research software. This means it's not ready for use as a router, but has reached the stage where the architecture is basically complete and it might be used as the basis for research. See the current status of the different parts of XORP, or the release notes for information about changes between releases.

  • Release 0.4, an alpha version of XORP, is available for download.
  • The XORP source code is accessible through our anonymous CVS server.
  • Individual files and changes to files can be viewed through a CVSweb interface.

Please note that to build all the XORP modules and test programs with debugging enabled, you will need nearly 1GB of free disk space. XORP is written in C++, and has been developed on FreeBSD 4.x and Redhat Linux 7.3. The current release 0.4 is known to compile with gcc 2.95.x, 2.96, and 3.2.1.
