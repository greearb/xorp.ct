What is Xorp?

Xorp (pronounced z-o-r-p)is an open router software platform being developed primarily at the International Computer Science Institute in Berkeley. The goal is to produce a complete set of router software that is stable enough and fully featured enough for production use, initially on edge routers, but also due to our focus on extensibility, to be an excellent platform for Internet research.

The following documents give more information:

  • Xorp goals and architecture (Sept 2002).
  • Software overview
  • Detailed source code documentation
  • Initial project proposal and design overview (Nov 2000).
  • Xorp is using the Click Modular Router as its primary forwarding path.

Xorp is free software. We are extremely grateful to Intel and the National Science Foundation for providing the initial funding to develop Xorp.

Getting Xorp

As of March 2003, Xorp is alpha release research software. This means it's not ready for use as a router, but has reached the stage where the architecture is basically complete and it might be used as the basis for research. See the current status of the different parts of Xorp.

  • Release 0.2, an alpha version of Xorp, is available for download.
  • The Xorp distribution is also available through our anonymous CVS server.

Please note that to build all the Xorp modules and test programs with debugging enabled, you will need nearly 1GB of free disk space. Xorp is written in C++, and has been developed on FreeBSD 4.x and Redhat Linux 7.2. The current release 0.2 is known to compile with gcc 2.95.x, 2.96, and 3.2.1.
